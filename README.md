# SunSeating App

Premium Attendance and Inventory Management System.

## 📱 Download the Android App (APK)
I have set up a **Cloud Robot** to build your APK for you!
1. **Push your code** to GitHub.
2. Go to the **[Actions]** tab at the top of your GitHub page.
3. Click on the latest **"Build Android APK"** run.
4. Download the **"SunSeating-App-APK"** artifact!

---

## 🚀 Deployment Instructions

1. **GitHub**: Push your code to your repository.
2. **Render**: Click the **"Deploy to Render"** button below to set up your database and server.

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/chandramondal-1/app)

---

## 💻 Local Preview
To run the app on your computer, use:
- `cd server && node index.js`
- `cd client && npm run dev`
